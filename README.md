# Shinobi Vision Pro

A real-time **Naruto-inspired hand gesture recognition system** built in **pure Python** using **OpenCV, MediaPipe, and NumPy**.

This project uses a webcam to detect hand landmarks, recognize gesture patterns, and trigger live jutsu-style visual effects in real time.

## Features

- Real-time webcam hand tracking
- Gesture-based jutsu triggering
- Multiple Naruto-inspired visual effects
- Lightweight Python implementation
- Built with computer vision and real-time rendering
